Implement Gmail SMTP (Nodemailer) OTP email delivery and wire it into the existing OTP endpoints.

- Keep existing OTP generation & storage logic; only replace console.log sending with real email sending.
- Add secure environment variables and update backend code accordingly.
- Provide a small test route or instructions to verify email delivery.

Environment variables (add to Replit Secrets / host env)

- SMTP_PORT=465
- SMTP_USER=<your-gmail-address@gmail.com>
- SMTP_PASS=<gmail-app-password>         # create an App Password in Google account; do NOT use your main Gmail password
- EMAIL_FROM="NexTravel <your-gmail-address@gmail.com>"  # sender shown in email

Files to create / modify
1. Create `server/src/lib/mail.js` (or `mail.ts` if using TS) with Nodemailer transport and helper `sendOtpEmail(to, code)`.

2. Modify the OTP request controller to call `sendOtpEmail(email, code)` instead of only console.log. The route used by frontend is `/api/auth/request-otp-email`. Keep same request/response shape.

3. Modify the OTP verification controller (`/api/auth/verify-otp-email`) to use email-based verification (no phone_number required). Ensure JWT generation includes email.

4. Add `.env.example` listing the env keys above (do not include real secrets).

Exact code to add (paste into files):

--- `server/src/lib/mail.js` ---
```js
const nodemailer = require('nodemailer');

const transporter = nodemailer.createTransport({
  host: process.env.SMTP_HOST || 'smtp.gmail.com',
  port: Number(process.env.SMTP_PORT) || 465,
  secure: true,
  auth: {
    user: process.env.SMTP_USER,
    pass: process.env.SMTP_PASS,
  },
});

async function sendOtpEmail(to, code) {
  const info = await transporter.sendMail({
    from: process.env.EMAIL_FROM,
    to,
    subject: 'Your NexTravel verification code',
    text: `Your NexTravel verification code is: ${code}`,
    html: `<p>Your NexTravel verification code is: <strong>${code}</strong></p><p>This code is valid for 2 minutes.</p>`,
  });
  return info;
}

module.exports = { sendOtpEmail };
